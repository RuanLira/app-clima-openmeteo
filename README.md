# App Clima

Um aplicativo de clima feito com HTML, CSS e JavaScript. O projeto usa a API da Open-Meteo para buscar cidades e exibir as condicoes atuais do tempo.

## Funcionalidades

- Buscar clima atual por cidade
- Exibir temperatura, sensacao termica, umidade e vento
- Mostrar descricao da condicao do tempo
- Salvar buscas recentes no navegador
- Tratar cidade nao encontrada e erros de busca
- Layout responsivo

## Tecnologias

- HTML
- CSS
- JavaScript
- Open-Meteo API

## Como usar

Abra o arquivo `index.html` no navegador ou publique o projeto no GitHub Pages.
